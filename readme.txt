The plugin allows you to display on any page of your blog movie showtimes. There is the possibility showtimes for any city.

== Description ==
The plugin is able to display on your blog movie showtimes for a given city, for 4 days, as there is a search string and the user can find your city. The plugin supports two languages, English and Russian (specified in settings).

Plugin their work uses a database service [Google Movies](http://www.google.com/movies "Google Movies")

Demonstration of the plugin can be found at http://pro-kino.com/today-in-the-cinema/
Official website of the plugin on my site http://tsepelev.ru/wp-showtime

== Installation ==

1. Unzip the `wp-showtime.zip` file.
1. Upload `wp-showtime` dir to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[showtime]` in your post or page body

== Frequently Asked Questions ==

= How do I show the film shows in my city? =

To do this, go to the plugin WP Showtime and in the `City or post code`, enter your location, such as `Moscow` or `London`.

= How can I apply their styles to display movie posters? =

To edit the appearance, you can use CSS, the class of `theater` - it displays information about the theater, the class of `kino` - it displays information about the film.
